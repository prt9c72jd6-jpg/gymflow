# 🏋️ GymFlow

> Você cuida dos alunos. O GymFlow cuida da gestão.

GymFlow é um sistema moderno de gestão para academias desenvolvido pela FlowTech.

Nosso objetivo é simplificar a administração de academias através de uma plataforma intuitiva, rápida e segura.

---

## 🚀 Recursos

- 👥 Gestão de alunos
- 💰 Controle financeiro
- 📅 Mensalidades
- 📈 Dashboard inteligente
- 📊 Relatórios
- 🔒 Segurança dos dados
